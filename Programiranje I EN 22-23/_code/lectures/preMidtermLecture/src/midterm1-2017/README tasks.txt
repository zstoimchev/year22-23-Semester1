    1 Midterm Exam Example

1. Write a program using Java programming language that reads two integer numbers 
    and writes their product to the screen!

2. Write a program using Java programming language that reads one integer number and outputs 
    (writes) the sum of all numbers that are lower or equal to this number and that are divisible by 5!

3. Write a program using Java programming language that reads an array of numbers and: 
    Program writes all even numbers!
    Program writes the number of all negative numbers!
    Program writes the product of all numbers on odd indexes in the array!
